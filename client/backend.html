Node.js Backend Layihə Quruluşu

Bu sənəd backend layihəsini sıfırdan necə düzgün qurmaq lazım olduğunu izah edir.

1. Layihəni Başlat

Görüləcək işlər:

  • npm init -y ilə layihəni başlat.
  • nodemon paketini devDependency olaraq quraşdır.
  • package.json daxilində scripts bölməsinə "dev": "nodemon server.js" əlavə et.

2. Lazımi Paketləri Quraşdır

Görüləcək işlər:

  • express, mongoose, dotenv, cors, helmet kimi əsas paketləri yüklə.
  • Əlavə ehtiyaclara uyğun joi, multer, bcryptjs, jsonwebtoken və s. quraşdır.

3. app.js və server.js

Görüləcək işlər:

  • app.js: Middleware-lər və route-lar burada qurulur.
  • server.js: Mongo bağlantısı edilir və server işə salınır.

4. config.js və db.js

Görüləcək işlər:

  • config.js: .env dəyişkənlərini mərkəzləşdir.
  • db.js: MongoDB bağlantı funksiyasını yaz.

5. schemas/ (Əgər validation istifadə edilirsə)

Görüləcək işlər:

  • İstifadəçi və digər məlumatlar üçün giriş yoxlamalarını bu fayllarda et.
  • Joi və ya Zod istifadə edilə bilər.

6. models/

Görüləcək işlər:

  • Mongoose schema və modellərini burada yarat.

7. services/

Görüləcək işlər:

  • Verilənlər bazasına aid funksiyalar burada yazılır.
  • Controller-lər bu servisləri istifadə edir.

8. controllers/

Görüləcək işlər:

  • Router-dən gələn sorğular burada idarə olunur.
  • Məlumatlar cavablandırılır, service funksiyaları çağırılır.
  • Əlavə errorHandler funksiyaları da burada ola bilər.

9. routers/

Görüləcək işlər:

  • Bütün route-ları Express Router ilə burada yaz.
  • Controller funksiyalarını çağır.

10. utils/

Görüləcək işlər:

  • Təkrar istifadə olunan yardımçı funksiyalar buraya yazılır.
  • Məsələn: Mongo məlumatlarını formatlamaq, mail göndərmək və s.

11. Əlavə Fayllar və Qovluqlar

Görüləcək işlər:

  • .env: Gizli məlumatlar burada saxlanılır (məs: DB URI).
  • .gitignore: node_modules və .env mütləq əlavə olunmalıdır.
